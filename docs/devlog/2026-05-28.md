## 1. Done（今日完成）

- B-141S 第一片：扩展 Vue 设置页 helper、`SettingsView` 和 `App.vue`，设置视图支持当前项目 Prompt 预设列表、内置模板复制、新建/编辑、删除、设置默认和清空默认；复用既有 `/api/prompt-presets*` 契约，不改后端 API 或数据库 schema。
- B-141T 第一片：新增 Vue 评估页 helper，扩展 `AssessmentView` 和 `App.vue`，评估视图支持开始评估、当前题作答、下一题/完成、结果概览、答题记录和待复测列表；复用既有 `/api/assessment/start` 与 `/api/assessment/answer` 契约，不改后端 API 或数据库 schema。
- B-141U 第一片：扩展 Vue 工作台回答 helper、`AnswerPanel`、`WorkbenchView` 和 `App.vue`，回答返回后支持提交“有用 / 无用 / 来源不准 / 需要更多上下文”四类本地反馈；复用既有 `/api/answer/feedback` 契约，不改后端 API 或数据库 schema。
- B-141V 第一片：新增 Vue 检索调试 helper 和 `SearchDebugPanel`，工作台支持提交检索诊断查询、临时调整 `top_k` / `min_score` / 关键词 / 向量参数，并展示来源质量、分块/向量状态和命中片段；复用既有 `/api/search/debug` 契约，不改后端 API 或数据库 schema。
- B-141W 第一片：新增 Vue Agent 工具 helper 和 `AgentToolsPanel`，工作台支持查看只读工具元数据、手动运行 `project_overview` / `search_sources`、查看工具结果、运行历史和单条详情；复用既有 `/api/agent/tools*` 契约，不改后端 API、工具白名单或数据库 schema。
- B-141X 第一片：扩展 Vue 回答 helper、`AnswerPanel`、`WorkbenchView` 和 `App.vue`，回答区支持展示 `tool_suggestion`、手动运行建议 `search_sources`、把成功工具结果标记为下一问 `tool_run_id` 上下文，并展示回答返回的 `tool_context`；复用既有 `/api/answer` 与 `/api/agent/tools/run` 契约，不改后端 API、工具白名单或数据库 schema。
- B-141Y 第一片：扩展 Vue 项目 helper、`SearchDebugPanel`、`WorkbenchView` 和 `App.vue`，工作台支持读取当前项目检索默认值，并把 `top_k` / `min_score` / 关键词 / 向量开关保存为项目默认值；复用既有 `GET/POST /api/projects/retrieval-settings` 契约，不改后端 API 或数据库 schema。
- B-141Z 第一片：扩展 Vue 检索 helper、`SearchDebugPanel`、`WorkbenchView` 和 `App.vue`，工作台支持保存检索复盘、查看当前项目复盘历史、读取单条详情和删除复盘记录；复用既有 `/api/retrieval/reviews*` 契约，不改后端 API 或数据库 schema。

- B-141U 只迁移 Vue 侧回答反馈按钮和保存状态，不迁移反馈备注输入、Workbench SSE/取消、聊天会话/历史、Agent 工具、检索调试或数据库 schema；原因是 `/api/answer/feedback` 后端契约已存在，本片可限定在工作台 UI 和 App 状态流。
- B-141V 只迁移 Vue 侧检索调试诊断入口，不迁移项目级检索默认值保存、检索复盘保存/列表/详情/删除、普通搜索结果、Workbench SSE/取消、聊天会话/历史、Agent 工具、检索算法或数据库 schema；原因是 `/api/search/debug` 后端契约已存在，本片可限定在工作台 UI 和 App 状态流。
- B-141W 只迁移 Vue 侧 Agent 只读工具手动运行面板，不迁移回答区工具建议按钮、工具来源回填到下一问、Workbench SSE/取消、聊天会话/历史、Agent 自动编排、工具白名单权限逻辑或数据库 schema；原因是 `/api/agent/tools*` 后端契约已存在，本片可限定在工作台 UI 和 App 状态流。
- B-141X 只迁移 Vue 侧工具建议与来源上下文，不迁移 Workbench SSE/取消、聊天会话/历史、Agent 自动编排、工具白名单权限逻辑、检索复盘、后端 API 或数据库 schema；原因是 `/api/answer` 已有 `tool_suggestion/tool_context/tool_run_id` 契约，B-141W 已提供手动运行只读工具能力。
- B-141Y 只迁移 Vue 侧项目级检索默认值读取/保存，不迁移检索复盘保存/列表/详情/删除、普通搜索结果、Workbench SSE/取消、聊天会话/历史、检索算法或数据库 schema；原因是 `GET/POST /api/projects/retrieval-settings` 后端契约已存在，B-141V 已提供检索调试参数控件。
- B-141Z 只迁移 Vue 侧检索复盘保存/列表/详情/删除，不迁移普通搜索结果、检索健康项目卡、Workbench SSE/取消、聊天会话/历史、评估题库/历史、检索算法或数据库 schema；原因是 `/api/retrieval/reviews*` 后端契约已存在，B-141V/B-141Y 已提供检索调试查询与参数控件。
