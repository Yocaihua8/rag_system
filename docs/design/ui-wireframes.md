# 页面线框与关键交互

> 状态：Active
> Owner：RAG 团队
> Last Updated：2026-05-16

## 1. 主界面结构

```text
+---------------------------+
| 左侧导航（项目空间/知识库/问答/生成/设置）|
| 1) 导航列表             |
+---------------------------+
| 右侧内容区 (QStacked)                |
| 2) 项目列表或导入页面                |
| 3) 文件列表与进度                    |
| 4) 问答输入与来源显示                |
| 5) 生成表单与结果                    |
+---------------------------+
| 底部任务状态栏：进度、警告、错误码       |
+---------------------------+
```

## 2. 关键流程

- **导入前**：无项目空间时提示创建并阻断问答按钮。
- **导入中**：显示 scan/process/index 进度分段。
- **问答中**：答案和来源列表同步刷新，支持错误重试。
- **首次启动**：首次引导页显示安装依赖和路径配置。

## 3. 文案统一

- 面向用户文案统一使用“项目空间”。
- 后台兼容字段不进入用户提示文案。
