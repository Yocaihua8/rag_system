## 2. Issues

- **问题**：Docker 模式下 Windows 路径无法直接挂载用于增量导入
  **影响**：用户须改用浏览器文件夹导入或手动挂载 `docker-workspace/`
  **当前处理**：B-59 浏览器导入覆盖主要场景；文档明确说明两种模式
  **跟踪**：见 `docs/guides/setup.md § 3`

## 3. Decisions

- 选用 Python 标准库 HTTP 服务而非 FastAPI → 原因：零额外依赖，Web MVP 快速可运行；后续 SSE 若受限再迁移（见 B-120）
- embedding 失败不阻断导入 → 原因：保证导入链路鲁棒性，本地 hashing 作为可用降级
