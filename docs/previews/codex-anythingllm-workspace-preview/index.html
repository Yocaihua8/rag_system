首页只保留当前工作区、会话列表、聊天内容、输入框、添加资料和依据入口。资料管理、导入历史、模型配置、练习和高级检查都不在首页展开。

这样用户进入应用后只需要做一件事：选择当前工作区，然后围绕资料继续提问。
