# 权限与边界说明

> 状态：Active
> Owner：RAG 团队
> Last Updated：2026-05-16

## 1. 当前权限模型

系统为单机单用户桌面客户端，核心权限边界为：

- 本地文件访问：用户当前登录凭据可访问目录。
- 本地配置访问：`runtime/` 与 `AppData` 目录下配置与数据库。
- 模型调用权限：取决于 Ollama/云端接口权限配置。

## 2. 风险级动作与限制

| 动作 | 权限要求 |
|------|----------|
| 读取本地文件 | 文件系统 ACL 许可 |
| 写入运行时数据库 | `runtime/` 目录可写 |
| 写入运行时向量库 | `runtime/vectors/` 目录可写 |
| 调用 API Key | 仅从配置来源读取，默认不回显明文 |

## 3. 现阶段未引入的机制

- 无角色权限（RBAC）与租户隔离；适用范围限定单机场景。
- 无 API 访问鉴权层（无外部服务入口）。
