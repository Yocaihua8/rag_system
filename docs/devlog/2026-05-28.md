- B-141V 第一片：新增 Vue 检索调试 helper 和 `SearchDebugPanel`，工作台支持提交检索诊断查询、临时调整 `top_k` / `min_score` / 关键词 / 向量参数，并展示来源质量、分块/向量状态和命中片段；复用既有 `/api/search/debug` 契约，不改后端 API 或数据库 schema。

## 3. Decisions

- B-141S 只迁移 Vue 侧 Prompt 预设管理入口，不迁移 Workbench SSE/会话、评估页、真实 LLM prompt 注入逻辑或数据库 schema；原因是 `/api/prompt-presets*` 后端契约和真实 LLM 注入行为已存在，本片可限定在设置页 UI 和 App 状态流。
- B-141T 只迁移 Vue 侧评估最小闭环，不迁移评估题库管理、历史评估列表、回答反馈、知识点画像、Workbench SSE/会话或数据库 schema；原因是评估开始/作答后端契约已存在，本片可限定在评估页 UI 和 App 状态流。
- B-141U 只迁移 Vue 侧回答反馈按钮和保存状态，不迁移反馈备注输入、Workbench SSE/取消、聊天会话/历史、Agent 工具、检索调试或数据库 schema；原因是 `/api/answer/feedback` 后端契约已存在，本片可限定在工作台 UI 和 App 状态流。
- B-141V 只迁移 Vue 侧检索调试诊断入口，不迁移项目级检索默认值保存、检索复盘保存/列表/详情/删除、普通搜索结果、Workbench SSE/取消、聊天会话/历史、Agent 工具、检索算法或数据库 schema；原因是 `/api/search/debug` 后端契约已存在，本片可限定在工作台 UI 和 App 状态流。
- 技术栈迁移整体完成前不推送；本片按计划继续本地提交。
