本地服务 DeepSeek

首次使用

  1. 先到设置页创建项目空间，或在资料库选择已有项目空间。
  2. 点击“选择本机文件夹导入”导入目录；Docker 用户不需要填写 Windows 路径。
  3. 导入完成后回到工作台提问、检索来源，或进入评估页开始检查题。
  4. 未配置模型时也可使用本地片段回答；需要真实模型回答时，在设置页配置 DeepSeek / OpenAI-compatible API Key。
