# 非技术用户发布清单（可勾选）

> 目标：让“没接触过计算机”的用户可以完成安装、启动、首次问答。

## A. 安装与启动

- [ ] 提供 Windows 安装包（双击安装）
- [ ] 安装后桌面有快捷方式
- [ ] 双击即可启动主程序（不需要命令行）
- [ ] 首次启动有引导提示（依赖/模型/项目空间）

## B. 首次使用引导

- [ ] 首次进入有 3 步引导（创建项目空间 -> 导入目录 -> 构建索引）
- [ ] 所有术语使用非技术描述（避免“向量化/索引分片”等）
- [ ] 每一步都提供“下一步”按钮
- [ ] 失败时有“怎么办”文字提示

## C. 错误与恢复

- [x] 无知识点时给出“未生成题目，请检查项目知识点是否已提取”
- [x] 题目不存在时给出“题目不存在，请重新开始评估”
- [x] 追问题目不存在时给出“未找到可复测题目，请重新开始评估后选择题目”
- [x] 评估提交前输入为空时不提交（仅客户端按钮与本地拦截）
- [ ] Ollama 未安装时给出安装提示
- [ ] 模型未下载时给出一键提示（推荐模型 + 命令）
- [ ] 无项目空间时禁用提问并提示先创建项目空间
- [ ] 关键错误可复制（方便远程支持）

## D. 功能可用性

- [ ] 可创建项目空间
- [ ] 可导入目录
- [ ] 可构建索引
- [ ] 可提问并看到回答
- [ ] 可查看引用来源
- [ ] 可查看问答历史
- [ ] 可生成文档并打开输出目录

## E. 文档与支持

- [ ] 提供《快速开始（1页）》
- [ ] 提供《常见问题（FAQ）》
- [ ] 提供《问题反馈方式》入口

## G. 当前版本发布收口（2026-05-18）

- [x] 评估链路闭环：开始评估 → 逐题提交 → 差距报告
- [x] 题目追问：根据 `follow_up_question_ids` 支持“继续追问”
- [x] 错题复测：`错题复测` 可触发错题优先出题，并支持无错题回退
- [x] 异常文案：会话失效、题目失效、错题缺失有明确可读提示
- [x] 回归测试通过：
  - `tests/test_application/test_knowledge_mastery_usecases.py`
  - `tests/test_desktop/test_project_knowledge_ui.py`

## F. 发布前验收

- [ ] 3 名非技术用户独立完成首次使用（不依赖开发者在场）
- [ ] 首次使用平均完成时间 < 10 分钟
- [ ] 关键路径失败率 < 10%
- [ ] 高优先级问题已修复
