# 知识库辅助管理页

> 状态：Active
> Owner：RAG 团队
> Last Updated：2026-06-29
> Related：docs/design/ui-wireframes.md, docs/design/api-spec.md, docs/features/frontend-engineering.md

## 1. 目标

知识库辅助管理页是 C 端用户日常维护单个知识库的入口。它不替代工作台问答，也不替代评估页的逐题作答流程，而是在资料库页顶部聚合当前项目的健康状态、摄入进度、文件列表入口、评估题库和最近结果。

## 2. 页面范围

- 展示当前项目状态：项目名称、文档数、chunk 数、向量数、聊天数、工具运行数、检索复盘数、最近活动时间。
- 展示检索健康：无 chunk、仅关键词可用、向量已生成三类状态。
- 展示摄入进度：最近导入批次的来源、状态、创建/更新/跳过/错误计数和完成时间。
- 展示文件列表入口：复用资料库现有文档列表、文档预览、集合筛选、导入和删除能力。
- 展示评估题库：当前项目已生成题目数和题型分布。
- 展示最近结果：当前项目已保存评估结果数和掌握状态分布。

## 3. 接口边界

- 项目状态复用 `GET /api/projects/summary?project_id=...`。
- 摄入进度复用 `GET /api/import/batches?project_id=...` 和 `GET /api/import/batches/detail?batch_id=...`。
- 文件列表复用 `GET /api/documents?project_id=...` 及现有文档集合接口。
- 评估题库和最近结果使用只读接口 `GET /api/assessment/library?project_id=...`；该接口返回题库数量、结果数量、题型分布、状态分布、题目快照和最近结果。

本功能不新增数据库表，不修改导入、检索、问答和评分算法。

## 4. 空状态与错误状态

- 未选择项目：显示“请选择项目空间后查看知识库管理概览”。
- 无文档：项目状态显示“知识库为空”，评估入口提示先导入文档。
- 无导入批次：摄入进度显示“暂无导入批次历史”。
- 无评估题目：题库显示“暂无评估题目”，引导用户到评估页开始评估。
- API 失败：对应卡片显示可恢复错误，不阻塞资料库其他区域。

## 5. 非目标

- 不实现批量文档操作、拖拽排序或文件夹树。
- 不实现跨刷新评估会话恢复。
- 不新增评估题目编辑、删除或手工创建题目。
- 不新增异步导入任务队列或实时进度 WebSocket。
- 不改 SQLite schema。
