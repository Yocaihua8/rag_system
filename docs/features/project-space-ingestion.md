# 项目空间与摄入

> 状态：Active
> Owner：RAG 团队
> Last Updated：2026-05-16
> Related ADR：无

## 1. 目标

将本地目录中文件稳定导入为可检索的知识资产，并在后续导入中仅处理变更文件。

## 2. 行为

- 首次导入：按支持后缀扫描并解析。
- 增量导入：仅对内容 checksum 变更文件重建 chunk/index。
- 文件删除：识别已删除文件并清理旧文档与检索内容。
- 失败容错：单个文件解析失败不中断批量任务。

## 3. 验收

- 修改文件后，源文件旧 chunk 不应残留。
- 删除文件后，对应来源不应在来源列表再出现。
- 删除前已导入的文件重启后不应影响其他文件查询。
