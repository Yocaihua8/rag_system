## 3. Decisions

- 导入批次不做回滚，不保存文档正文 → 原因：第一版聚焦可查询历史；完整回滚仍需单独评估，不能等同于项目备份恢复
- 文档集合只保存关联关系，删除集合不删除文档 → 原因：避免误删文档，集合是虚拟分组
